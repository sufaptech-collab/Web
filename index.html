A description is a verbal or written representation of a thing's or person's characteristics. To describe something well, observe its details like size, shape, color, and texture, then use precise and descriptive language to convey those observations. For example, instead of saying a mug is a "mug," a good description would be that it is a "cylindrical, deep matte blue mug with a smooth texture and a thick, easy-to-grip handle
